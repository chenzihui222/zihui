# Zihui Chen Portfolio

Zihui Chen 的个人简历和项目主页，使用 Astro + Tailwind CSS 构建。

## 🚀 项目结构

```
/
├── src/
│   ├── components/
│   │   ├── Hero.astro
│   │   ├── About.astro
│   │   ├── Projects.astro
│   │   ├── Skills.astro
│   │   ├── Contact.astro
│   │   └── Footer.astro
│   ├── layouts/
│   │   └── Layout.astro
│   └── pages/
│       └── index.astro
├── public/
├── astro.config.mjs
├── tailwind.config.mjs
└── package.json
```

## 🧞 命令

所有命令都在项目根目录的终端运行：

| 命令                   | 动作                                           |
| :--------------------- | :--------------------------------------------- |
| `npm install`          | 安装依赖                                       |
| `npm run dev`          | 启动本地开发服务器 `localhost:4321`            |
| `npm run build`        | 构建生产版本到 `./dist/`                       |
| `npm run preview`      | 在部署前本地预览构建                           |

## 📦 技术栈

- **框架**: [Astro](https://astro.build)
- **样式**: [Tailwind CSS](https://tailwindcss.com)
- **语言**: TypeScript
